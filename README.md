# extory-blog 과제
# React + TypeScript + Vite + Tailwind

## 작업기간
2025.5.19 ~ 2025.5.20

************************************
## **1. 프로젝트 클론**
$ git clone https://github.com/pepperflavor/extory-blog.git
$ cd extory-blog

## **2. 의존성 설치**
$ npm isntall

## **3. 개발 서버 실행**
$ npm run dev

 또는
$ yarn dev

## **4. 프로덕트 빌드**
$ npm run build

## **+) 환경변수 추가**
.env
VITE_API_BASE_URL = https://extory.co/api
